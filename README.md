# Binarq System Management

> Plataforma web completa diseñada para optimizar la gestión operativa de empresas con múltiples proyectos o sedes. Ofrece módulos especializados para la administración de recursos humanos, control de inventarios y gestión de soporte técnico, con un robusto sistema de autenticación, roles y permisos.

## Características Principales

#### Gestión de Recursos Humanos (HR)
- **Empleados**: Registro completo con datos personales, laborales y documentación
- **Planillas**: Generación y gestión de nóminas con cálculos automáticos
- **Prestaciones**: Cálculo automatizado de prestaciones laborales
- **Vacaciones**: Control de solicitudes y períodos vacacionales
- **Contratos**: Gestión documental de contratos laborales

#### Gestión de Inventarios y Bodegas (WH)
- **Materiales**: Control de stock con alertas de mínimos
- **Herramientas**: Seguimiento de ubicación y estado
- **Solicitudes**: Sistema de requisición entre proyectos con flujo de aprobación
- **Mantenimiento**: Programación y registro de mantenimientos preventivos/correctivos
- **Categorización**: Organización flexible de items

#### Soporte Técnico y Desarrollo (TI)
- **Tickets**: Sistema de tickets con prioridades y asignaciones
- **Proyectos**: Gestión de proyectos y asignación de personal
- **Historial**: Seguimiento completo de incidencias

#### Sistema de Seguridad
- Autenticación con JWT (cookies seguras)
- Control de acceso basado en roles (RBAC)
- Permisos por servicio y proyecto

#### Funciones Transversales
- **Notificaciones**: Sistema de alertas por correo electrónico
- **Reportes**: Generación de PDFs para documentos y reportes
- **Multi-proyecto**: Soporte para múltiples sedes/proyectos
- **Auditoría**: Logs de sistema y seguimiento de eventos
